What is Matchups?

Matchups is a new way for people to collectively contribute to an overall ranking of anything. This is done by giving each item a base rating and using the elo rating system to compare them. This comparison calculates the expected outcome based on both ratings and changes both accordingly. Performing this calculation with many user's choices will (hopefully) yield accurate rankings of everything.

Matchups is built upon users adding lists and items. Go HERE to see the lists of items and go HERE to add a list.
